# Vishal Patil - Professional Data Portfolio 📊

A personal portfolio website designed to showcase my background, skills, and projects as a Data & Insight Analyst. This interactive site highlights my experience bridging the gap between complex data engineering and executive decision-making.

## 🚀 Key Features

*   **Responsive Design:** Fully responsive layout optimized for all devices (desktop, tablet, mobile), ensuring a seamless viewing experience.
*   **"ZeeFrames" Aesthetic:** Clean, modern UI focusing on professionalism and readability.
*   **Project Showcase:** Dedicated sections for highlighting impactful analytical and engineering projects (e.g., Arrears & Income Risk Predictor, Omnichannel Strategy).
*   **CV Download:** Easy access for recruiters/hiring managers to download my latest professional CV immediately.
*   **Smooth Navigation:** Seamless scrolling menu and mobile hamburger navigation for effortless exploration.

## 🛠️ Tech Stack

This project was built without heavy frameworks to ensure lightning-fast performance and clean code.

*   **HTML5:** Semantic structure for optimal accessibility and SEO.
*   **CSS3:** Custom styling (Flexbox/Grid), variables for theming, and smooth transitions/animations.
*   **Vanilla JavaScript (ES6+):** DOM manipulation, smooth scrolling, and mobile menu interactivity.

## 📁 Repository Structure

*   `index.html`: The main structure and content of the portfolio.
*   `style.css`: All styling rules, branding colors, and responsive media queries.
*   `script.js`: Interactive elements, event listeners, and dynamic UI behavior.
*   `Vishal_Patil_CV.pdf`: Up-to-date downloadable resume.
*   `avatar.png`: Profile image for the hero section.

## 💻 Local Setup

No build tools or Node.js required. To view the project locally:

1.  Clone this repository:
    ```bash
    git clone https://github.com/your-username/vishal-portfolio.git
    ```
2.  Navigate to the directory:
    ```bash
    cd vishal-portfolio
    ```
3.  Open the `index.html` file in your preferred web browser. You can do this by double-clicking the file in your file explorer, or using an extension like VS Code's "Live Server".

## 🌐 Deployment

The site is configured to be deployed easily via **GitHub Pages**.

1.  Push the code to the `main` branch.
2.  Go to the repository **Settings** > **Pages**.
3.  Under Source, select the `main` branch and `/ (root)` folder, then save.
4.  Your site will be live at `https://your-username.github.io/vishal-portfolio/`.
